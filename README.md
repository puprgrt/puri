Landing Page Pusat Pelayanan Publik - Dinas PUPR Kabupaten GarutRepositori ini berisi kode sumber untuk Landing Page Pusat Pelayanan Publik terpadu dari Dinas Pekerjaan Umum dan Penataan Ruang (PUPR) Kabupaten Garut.Website ini dirancang untuk memberikan kemudahan akses informasi dan layanan bagi masyarakat terkait infrastruktur dan penataan ruang, dengan antarmuka yang modern, profesional, dan responsif.🎨 Tema DesainAntarmuka dibangun dengan mengusung gaya Modern Glassmorphism, memadukan elemen-elemen desain berikut:Warna Utama: Biru tua pekat (Navy/Dark Blue) merepresentasikan profesionalisme dan ketegasan institusi.Warna Aksen: Kuning Khas PU dan Biru Muda (Cyan/Light Blue) untuk menyoroti elemen interaktif dan Call-to-Action.Efek Visual: Pendaran cahaya (glow effect) di belakang backdrop tembus pandang (glass effect) untuk memberikan kesan futuristik namun tetap elegan.✨ Fitur UtamaHalaman pendaratan ini menyediakan akses cepat ke lima layanan utama Pusat Pelayanan Publik PUPR Garut:Penilaian Kerusakan (Telah terhubung dengan SI-PEKA)Evaluasi komprehensif, audit fisik, dan dokumentasi tingkat kerusakan bangunan gedung atau infrastruktur lainnya.Bantek Pengecekan PerencanaanBantuan teknis untuk pengecekan detail desain, struktur, dan kesesuaian dokumen perencanaan pembangunan.Bantek Pelaksanaan KonstruksiPengawasan dan bantuan teknis selama masa pelaksanaan proyek untuk memastikan kualitas konstruksi sesuai standar.Rekomendasi Kebutuhan AnggaranAnalisis dan penyusunan rekomendasi estimasi biaya yang tepat sasaran untuk perbaikan atau pembangunan infrastruktur.Assessment (Asesmen)Kajian kelayakan, kekuatan struktur, dan fungsi bangunan guna memastikan keselamatan dan kepatuhan standar.🚀 Teknologi yang DigunakanProyek ini dibangun menggunakan teknologi web standar tanpa framework JavaScript yang berat, sehingga sangat cepat dimuat:HTML5: Struktur semantik halaman.Tailwind CSS (via CDN): Framework CSS berbasis utilitas untuk penataan gaya yang cepat dan responsif.Phosphor Icons: Pustaka ikon yang bersih dan modern.Google Fonts (Plus Jakarta Sans): Tipografi yang modern dan mudah dibaca.🛠️ Instalasi & PenggunaanKarena ini adalah halaman HTML statis tunggal, tidak diperlukan proses build yang rumit.Clone repositori:git clone https://github.com/puprgrt/puri.git
Buka file:Buka folder hasil clone dan cukup klik dua kali pada file index.html untuk menjalankannya di browser Anda (Chrome, Firefox, Safari, dll).Modifikasi:Buka file index.html dengan code editor pilihan Anda (seperti VS Code, Sublime Text, atau Notepad++) untuk memperbarui tautan (link) layanan, teks, atau gambar.🔗 Tautan PentingSistem Penilaian Kerusakan (SI-PEKA): https://si-peka-820728979520.asia-southeast1.run.app/📞 Kontak & DukunganDinas Pekerjaan Umum dan Penataan Ruang Kabupaten Garut📍 Jl.
